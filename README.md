# Żeby spróbować zagrać w naszą grę, musicie mieć zainstalowane Node.js w wersji 20.0.0 lub wyższej, a także npm w wersji 9.0.0 lub wyższej.

# Aby zainstalować i uruchomić grę, należy wykonać poniższe kroki:

## 1. git clone https://github.com/ADONYARIIK/Error_404_Team_Hackathon2025

## 2. npm install

## 3. npm run dev

## 4. Otworzyć w przeglądarce podany link do localhosta.
